## Features Available

### Admin Features
- Dashboard with key metrics
- Inventory management (add items, stock in, adjust)
- Test management with BOM recipes
- Reports (sales and inventory)
- User management

### Doctor Features
- Dashboard with today's visits
- Patient management (add, search, view history)
- Create visits and consultations
- Order lab tests (auto-deducts consumables)
- Dispense medicines (auto-deducts stock)
- Generate and print receipts

## Testing the System

1. Login as doctor
2. Create a new patient or select existing
3. Create a visit
4. Add a lab test (e.g., CBC) - observe stock deduction
5. Dispense medicine (e.g., Paracetamol) - observe stock deduction
6. Set payment details
7. Save visit and view receipt
8. Login as admin to view reports and stock movements
